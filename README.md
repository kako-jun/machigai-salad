# 🦐 小エビの間違いサラダ (Machigai Salad)

**Shrimp Salad of Differences**

サイゼリヤの人気コンテンツ「間違い探し」をスマートフォンなどのブラウザ上で解決するための、完全クライアントサイド型Webアプリです。

## 🎯 特徴

- **完全プライバシー保護**: 画像は一切アップロードされず、すべてブラウザ内で処理
- **コストゼロ**: 外部APIや従量課金のAIサービスは使用しない
- **簡単操作**: 写真を撮るだけで自動的に左右を分割して比較
- **タップで切り替え**: タップしている間だけ画像が切り替わる直感的なUI

## 🚀 技術スタック

- **Next.js** (静的エクスポート) - React フレームワーク
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング
- **OpenCV.js** - 画像処理（色調補正、分割）
- **GitHub Pages** - テスト環境ホスティング
- **Cloudflare Pages** - 本番環境ホスティング（将来）

## 📱 使い方

1. アプリを開く
2. 間違い探しの画像を撮影または選択
3. アプリが自動的に左右に分割・色調補正
4. 画面をタップして画像を切り替えて比較
5. 間違いを見つける！

## 🛠️ 開発

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

http://localhost:3000 でアプリが起動します。

### ビルド

```bash
npm run build
```

静的ファイルが `out/` ディレクトリに生成されます。

## 📦 デプロイ

### GitHub Pages (テスト環境)

mainブランチまたはclaude/\*ブランチにpushすると、自動的にGitHub Pagesにデプロイされます。

設定が必要:

1. リポジトリの Settings → Pages
2. Source を "GitHub Actions" に設定

### Cloudflare Pages (本番環境・将来)

将来的にCloudflare Pagesでホストする予定です。

## 🧪 画像処理の流れ

1. ユーザーが画像を撮影/選択
2. OpenCV.jsで画像を読み込み
3. 色調補正（ヒストグラム均等化）を実施
4. 画像を中央で左右に分割
5. 分割した画像をキャンバスに表示
6. ユーザーがタップで切り替え

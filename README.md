# 小エビの間違いサラダ (Machigai Salad)

紙に印刷された「間違い探し」をスマートフォンなどのブラウザ上で解決するための、完全クライアントサイド型Webアプリです。

**https://machigai-salad.llll-ll.com**

## 特徴

- **完全プライバシー保護**: 画像は一切アップロードされず、すべてブラウザ内で処理
- **コストゼロ**: 外部APIや従量課金のAIサービスは使用しない
- **簡単操作**: 写真を撮るだけで自動的に左右を分割して比較
- **タップで切り替え**: タップしている間だけ画像が切り替わる直感的なUI

## 使い方

1. アプリを開く
2. 間違い探しの画像を撮影または選択
3. 紙の角を自動検出（手動調整も可能）
4. 台形補正して左右に分割
5. 画面をタップして画像を切り替えて比較
6. 間違いを見つける！

## 技術スタック

- **Next.js** (静的エクスポート) - React フレームワーク
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング
- **OpenCV.js** - 画像処理（台形補正、分割）
- **Cloudflare Pages** - ホスティング（GitHub 連携で自動デプロイ）

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動（http://localhost:3000）
npm run build    # ビルド（out/ に静的ファイル生成）
npm run lint     # Lint 実行
```

## デプロイ

Cloudflare Pages の GitHub 連携で `main` ブランチへの push 時に自動デプロイ。

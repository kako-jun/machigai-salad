# 使い方ガイド

## 基本の流れ

操作は大きく3ステップ。「撮る → 調整する → 比べる」だけ。

---

## ステップ1：撮影する

アプリを開くと、カメラボタンと画像選択ボタンが表示される。

**スマートフォンで撮る場合**

- カメラボタンをタップすると背面カメラが起動する
- ランチョンマットや紙をテーブルに置いた状態で、真上から撮るとうまくいく
- 斜めになっても構わない。台形補正で自動的に正面向きに直せる
- 明るい場所で撮ると認識精度が上がる

**すでに撮った写真を使う場合**

- 画像選択ボタンをタップしてギャラリーから選ぶ

**PCで使う場合**

- 「ファイルを選択」からパソコンに保存した画像を選ぶ

---

## ステップ2：角を調整する（任意）

画像を読み込むと、紙の4つの角が自動的に検出される。
オレンジ色の丸が4箇所に表示されるので、ずれていたら指でうごかして調整する。

**ルーペ（拡大鏡）**

角をうごかしている間、画面の反対側にルーペが表示される。
指で隠れている部分が3倍に拡大され、十字線で正確な位置がわかる。

**検出感度の切り替え**

紙がうまく検出されない場合、「もう1回けんしゅつ」ボタンをタップすると検出感度を切り替えて再検出できる。
感度は「きびしめ → ふつう → ゆるめ」の3段階で順に切り替わる。撮影環境に合った感度を試してみよう。

**中央の破線**

角調整の画面には白い破線が表示されている。これは左右に分割するときの境界線。
角をうごかすとリアルタイムに破線も動くので、分割位置を確認しながら調整できる。

**うまくいくコツ**

- 4つの丸が紙の四隅と重なるように合わせる
- 斜めから撮った場合は台形に配置されることがある。それで正しい
- ルーペの十字線を紙の角に合わせると正確
- 細かい調整が難しければ「そのままでOK」を選んでスキップできる

**この調整で何が変わるか**

- 斜めに撮った写真が、正面から見た形に変換される
- 正確に合わせるほど、左右の分割が正確になる

---

## ステップ3：比較する

2枚の画像が重なった状態で表示される。

**基本操作**

- 何もしない状態：左の絵（1枚目）が表示されている
- 画面を長おしする：右の絵（2枚目）に切り替わる
- 指を離す：1枚目に戻る

**スライドで位置微調整**

左右の画像が少しずれている場合、角調整に戻らなくてもその場で直せる。

- 指を置いて少し動かすと、スライドモードに切り替わる
- スライド中は左の絵が半透明になり、右の絵が透けて見える
- 重なり具合を見ながら位置を合わせる
- 指を離すと確定。「かどの調整にもどる」ボタンで角調整フェーズに戻れる

**コーナーワープで形を合わせる**

左の画像の四隅にオレンジ色のハンドルが表示されている。

- ハンドルを指でドラッグすると、左の絵が歪んで右の絵の形に合わせられる
- ドラッグ中は左の絵が半透明になり、右の絵との重なりを確認しながら調整できる
- 4つの角をそれぞれ独立に動かせるので、斜めのずれにも対応できる

**アンドゥ（もどす）**

角調整画面・比較画面の両方に「もどす」ボタンがある。
ドラッグ操作を1回分取り消して、直前の状態に戻せる。

**比較結果をシェアする**

比較画面の「アニメーションをシェア」ボタンをタップすると、左右の絵が1秒ごとに切り替わるアニメーションGIFが作られる。
スライドや四隅の調整もGIFに反映されるので、自分が見ている状態そのままをシェアできる。
作られたGIFはLINEやXなどで友達に送れる。「この違い、わかる？」とシェアして楽しめる。

**使い方のコツ**

- 怪しいと思った箇所を見ながらタップ・リリースを繰り返す
- 上部のバーに「ひだり / みぎ」が表示されるので、今どちらを見ているかわかる
- ゆっくりタップすると変化がよくわかる

## よくある質問

**「じゅんび中...」と表示されてボタンが押せない**
画像処理の準備中。数秒待つと使えるようになる。

**紙が検出されなかった**
角調整の画面が出ずに比較画面に進む場合は、自動検出できなかったケース。
そのまま使えるが、斜め補正はかかっていない状態になる。
より明るい場所で、紙全体が画面に収まるように撮り直すと改善することが多い。

**画像が斜めのまま比較画面になった**
角調整ステップで「そのままでOK」を選んだか、自動検出に失敗した状態。
「別の絵で遊ぶ」ボタンで最初に戻り、撮り直して再挑戦できる。

**PCのマウスでも使える？**
使える。マウスボタンを押している間は2枚目が表示され、離すと1枚目に戻る。

## 保存と復元

比較画面に「ほぞん」ボタンがある。
タップすると、今の状態（元画像・角の調整・位置ずらし・コーナーワープ）がスマートフォン/PCのブラウザに保存される。

同じ画像で何度「ほぞん」を押しても、前の保存を上書きする（保存枠を消費しない）。
新しい画像を読み込んだときだけ、次の保存は新しい枠に入る。

**保存したデータを開く**

- トップ画面に「ほぞんしたやつ (件数)」ボタンが表示される
- タップするとポップアップで一覧が表示される
- 日時をタップすると、その時の状態に戻って再比較できる
- ゴミ箱アイコンで不要なデータを削除できる

**注意**

- 保存できるのは最大5つまで。いっぱいになると古いものから自動で消える
- データはブラウザのLocalStorageに保存される。ブラウザのデータを消すと消える
- 別のブラウザやデバイスとは共有されない
